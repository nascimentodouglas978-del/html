Tags Semanticas em HTML 5

As tags semanticas são elementos HTML que descrevem claramente o significado do conteudo que elas envolvem. Elas ajudam a melhorar a acessibilidade, a SED e a estrutura do documento. Aqui estão algumas das principais tags semanticas:

  • <header>: Define o cabeçalho de um documento ou seção.
  • <nav>: Define um conjunto de links de navegação.
  • <main>: Define o conteudo principal do documento.
  • <section>: Define uma seção generica de um documento.
  • <article>: Define um conteudo independente e auto-contido , como um artigo de blog ou noticia.
  • <aside>: Define um conteudo relacionado, mas separado do conteudo principal, como uma barra lateral.
  • <footer>: Define o rodapé de um documento ou seção.

usar essas tags semanticas ajuda os motores de busca a entender melhor o conteudo do seu site e melhora a experiencia do usuario , especialmente para aqueles que utilizam leitores de tela.
